# Vendored brand fonts (SIL Open Font License 1.1)

Self-hosted woff2 for the attack-flow timeline emitter, inlined as data URIs so the
emitted HTML is self-contained and offline (no CDN, CSP-safe). All faces are OFL 1.1
and redistributable.

- Archivo Narrow (display), Archivo (labels), Inter (body): https://github.com/Omnibus-Type
- JetBrains Mono (data/timestamps): https://github.com/JetBrains/JetBrainsMono
- Caveat (annotation): https://github.com/googlefonts/caveat

Roles per docs/brand.md typography table.
